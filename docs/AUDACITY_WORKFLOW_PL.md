# Workflow z Audacity

Audacity traktujemy jako edytor pośredni:

1. Program eksportuje `prepared_for_audacity.wav`.
2. Otwierasz plik w Audacity.
3. Możesz użyć makr, normalizacji, prostego EQ, cięcia i ręcznej edycji.
4. Eksportujesz wynik jako WAV.
5. Program kontynuuje pipeline: Melodyne/RX/mastering/voice conversion.
